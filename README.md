UrbanVault.
A UrbanVault é uma marca originalmente portuguesa, criada por Rafael Feijão, neste projeto criarei o site da minha loja, usando como linguanges:
-PHP;
-JavaScript;
-CSS;
-HTML;
-PHP MY ADMIN.
O site será completo com tudo o que uma loja online precisa de ter, desde login, carrinho, novidades, todos os produtos disponiveis até ao método de pagamento.
